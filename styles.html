Styles

Flat

All UI design follows Flat design, a minimalistic UI design genre that is adopted by all major mobile OS.

Specifically, we consider shadow to be the last solution when create visual hierarchy. Therefore, please use shadow very prudently and sparsely, and only out of absolute necessity.

We delute the definition of the border of UI containers and eliminate any unnecessary decoration, including getting rid of seams in the list and only apply seams when you can categorize the items inside the list. By doing so, we create a very sublte styling and allow the content to take a more dominent position in the visual hierarchy.

The ultimate goal of this UI styling is to create a immersive learning environment.
